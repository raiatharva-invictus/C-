/*
IMP NOTES:
1 we can instead of defining mutiple classes in a single file we can distribute the code and use it here via include : using header file 
2 in header file we can only write prototype of the function and constructors and separate the implementation
3 the name of header file and implementation file does not need to be same but it is better practice and some compilers are optimized this way to assume this convention(of same name of header and implementation file(cpp file that has the implementation of the fucntion prototyped in header file))
*/
